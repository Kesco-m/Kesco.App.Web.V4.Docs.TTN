Заполнение документа

Форма ТТН состоит из трех вкладок "Реквизиты накладной", "Товары" и "Услуги".

Навигация по данным вкладкам, а также по вкладкам внутри вкладки "Реквизиты накладной" осуществляется с помощью клавиш со стрелками.

Поля помеченные желтым цветом обязательны для заполнения.

  • «Номер документа» - поле содержит номер ТТН (для автоматической генерации номера нажмите на "сгенерировать");
  • «Дата» - указывается дата ТТН;
  • «Документ корректировочный» - указывает, что данный документ является корректировочным по отношению к документу, выбранному в поле "Корректировочный документ";

    При выборе корректируемого документа учитываются следущие ограничения: тип документа (Товарно-транспортная накладная), связанный договор, лица-контрагенты. Если возникает цепочка документов Основной документ --> Корректирующий1 --> Корректирующий2, то при выборе в качестве корректируемого документа любого кроме последнего, будет выдано сообщение о том, что выбранный документ был скорректирован с помощью Корректирующий2 (в данном случае он последний в цепочке) и предложением выбрать его в качестве изменяемого. Т.е. документ корректируется только при помощи одного документа!

Реквизиты накладной

Данные накладной

  • «Дата проводки» - поле содержит дату проводки документа (по умолчанию подставляется дата из поля «Дата»);
  • «Валюта оплаты» - валюта взаиморасчетов;
  • «Примечание» - дополнительная информация для печатных форм;

Грузоотправитель

  • «Грузоотправитель» - название грузоотправителя;
  • «Пункт отправления» - пункт отправления;
  • «Отметки грузоотправителя» - дополнительная информация
  • «Реквизиты грузоотправителя» - приводит к форме выбора адреса из доступных адресов (можно ввести адрес вручную);
  • «Р/С грузоотправителя» - расчетный счет грузоотправителя, выбирается из справочника;

название грузополучателя;

  • «Грузополучатель» - грузополучатель
  • «Пункт назначения» - пункт назначения;
  • «Отметки грузополучателя» - дополнительная информация
  • «Реквизиты грузополучателя» - приводит к форме выбора адреса из доступных адресов (можно ввести адрес вручную);
  • «Р/С грузополучателя» - расчетный счет грузополучателя, выбирается из справочника;

Поставщик

  • «Совпадает с грузоотправителем» - указывается в том случае, если поставщик и грузоотправитель одно лицо.
  • «Поставщик» - служит для выбора лица поставщика.

    При вводе названия лица с клавиатуры и нажатии enter, происходит контекстный поиск лица в справочнике, и выдаются подходящие варианты. Если не найдено ни одного лица, программа предложит расширенный поиск по справочнику лиц, или создать лицо.

  • «Адрес поставщика» - автозаполняется в случае, если определен юридический адрес, иначе, нажатие на кнопку выбора адреса приводит к форме выбора адреса из доступных адресов (можно ввести адрес вручную);
  • «Название поставщика» - содержит название лица;
  • «ОКПО поставщика» - ОКПО лица, поле либо автозаполняется, либо возможен ввод вручную;
  • «Р/С поставщика» - расчетный счет поставщика выбирается из справочника среди счетов, валюта которых совпадает с валютой ТТН (если заполнено поле Поставщик, то данные Р/С выбираются среди счетов Поставщика, если поле Поставщик не заполнено, то выбор Р/С происходит среди всех действующих счетов, с помощью расширенного поиска); Если у Поставщика существует единственный счет в выбранной валюте взаиморасчетов, поле Б/С Поставщика заполняется автоматически. Если поле Б/С Поставщика было очищено, то поле Поставщик остается без изменений, а поле Банковские реквизиты Поставщика очищаются.
  • «ФИО руководителя»
  • «Должность руководителя»
  • «ФИО бухгалтера»
  • «Должность бухгалтера»
  • «Отпуск груза произвел»
  • «Отпуск груза произвел должность»

Плательщик

  • «Совпадает с грузополучателем» - указывается в том случае, если плательщик и грузополучатель одно лицо
  • «Плательщик» - служит для выбора лица плательщика;
  • «Адрес плательщика» - адрес плательщика
  • «Название плательщика» - содержит название лица;
  • «ОКПО плательщика» - ОКПО лица, поле либо автозаполняется, либо возможен ввод вручную;
  • «Р/С плательщика» - расчетный счет плательщика

Документы

  • «Договор» - поле содержит название (номер договора) между Поставщиком и Плательщиком.

    Если предварительно заполнены поля «Поставщик» или «Плательщик», то поиск договора производится только по указанным лицам, в случае наличия единственного договора, поле заполоняется автоматически. Есть возможность ввести в строку Договор код или номер договора, и если такой документ найден, то поле заполняется. Если поля «Поставщик» или «Плательщик» не заполнены, то при выборе Договора произойдет автоподстановка в эти поля значений из договора. Если у договора есть не аннулированные изображения с факсимиле, то слева от контрола выбора договора рисуется иконка штампа. Если навести мышкой на иконку, то появляется список, в котором собраны все НЕ ПОВТОРЯЮЩИЕСЯ штампы, со всех НЕ АННУЛИРОВАННЫХ изображений. Если штампы повторяются, то будет только одна запись каждого штампа. Если изображение аннулировано, то иконка штампа не рисуется, тултипа нет.

  • «Приложение» - поле служит для выбора приложения к договору;

    Если предварительно выбран Плательщик или Получатель, то поиск документа происходит по этим лицам. Приложение выбирается только среди Приложений к выбранному договору (при заполненном поле Договор). Есть возможность ввести в строку Приложение к договору код или номер приложения, и если такой документ найден, то поле заполняется. Если поле приложение заполнено, то договор выбирается автоматически (при не заполненном поле Договор). В случае, наличия у договора единственного приложения, то при выборе договора, поле приложение заполняется автоматически.

  • «Заявка на покупку»
  • «Аккредитив»
  • «Коносамент»
  • «Счет, Инвойс проформа»
  • «Платежные документы»

Транспорт

  • «Номер автомобиля»

Управляющие кнопки.

  • «Сохранить» - сохраняет текущую форму
  • «Обновить» - обновляет поля формы
  • «Копировать» - очищает все поля формы

Товары

Список позиций товаров по ТТН. Данный список можно группировать по порядку ввода, товарам или вагонам. Для этого служат соответствующие управляющие кнопки выбора вида отчета:

  • «Реализация по товарам»
  • «Реализация товаров по вагонам»
  • «Сгруппировать по товару»

Для каждой колонки сформированного списка можно задать сортировку и установить фильтр. Для этого неоходимо нажать на иконку справа от названия колонки.

Управляющие кнопки располагаются в первой колонке. Они позволяют скопировать, удалить или отредактировать позицию, если запись находится в режиме редактирования.

  • «Месяц ресурсов»
  • «Склад поставщика» - склад поставщика для всех товаров. При изменении склад изменится у всех товаров данной ТТН.
  • «Склад плательщика» - склад плательщика для всех товаров. При изменении склад изменится у всех товаров данной ТТН.

Управляющие кнопки.

  • «Добавить товар» - добавить новую позицию товара (горячая клавиша "Insert").
  • «Выбрать участки отправок вагонов» - открывает форму поиска и выбора вагонов

Заполнение данных о товаре

  • «Склад поставщика» - склад поставщика заявленного ресурса;
  • «Склад плательщика» - склад плательщика заявленного ресурса;
  • «Товар» - Содержит название заявленного товара.
  • «Название товара (рус)» - русское название товара;
  • «Название товара (лат)» - латинское название товара;
  • «Выбор ед.изм» - поле служит только для выбора единицы измерения (данные в данном поле не сохраняются);
  • «Ед. изм.» - единица, в которой измеряется заявленное количество, поле заполняется автоматически при выборе "Выбор ед.изм";
  • «Эквивалент» - выводит формулу пересчета дополнительной ед.изм. в основную, поле заполняется автоматически;
  • «Страна происхождения» - место производства товара;
  • «ТД» - документ таможенной декларации;
  • «Количество» - числовое поле, содержит данные по количеству товара;
  • «Цена без НДС» - содержит цену за единицу товара без учета налога НДС;
  • «Ставка НДС» - выбирается процентная ставка налога НДС;
  • «Сумма без НДС» - содержит суммарную стоимость по позиции без учета налога НДС;
  • «НДС» - сумма налога НДС;
  • «Акциз» - сумма акциза, вводится вручную;
  • «Всего» - суммарная стоимость по позиции с учетом налога НДС;
  • «Валюта» - поле просмотра валюты;

При вводе или изменении данных в полях «Сумма без НДС», «НДС», «Всего» может появляться окно «Выбор типа пересчета», предназначенное для выбора нужного типа пересчета значений для данных полей.

Для изменения и пересчета сделанных изменений по позиции нажмите кнопку «ОК», для обратного пересчета данного значения нажмите кнопку «Пересчет», для отмены внесенных изменений «Отмена», кнопка «Изменить» приводит к принудительному изменению значения без пересчета.

При вводе или изменении данных в полях «Цена без НДС»,«Ставка НДС» может появляться окно «Выбор вида расчета», предназначенное для уточнения введенной суммы - с НДС или без.

Управляющие кнопки.

  • «Сохранить» - сохраняет текущую форму позиции товара (горячая клавиша "F2")
  • «Обновить» - обновляет поля формы
  • «Удалить» - удаляет текущую форму позиции товара
  • «Закрыть» - закрывает форму без сохранения
